Este projeto tem como objetivo apresentar conceitos fundamentais de **saúde financeira** para iniciantes.  
A ideia é ajudar pessoas a entenderem como organizar suas finanças pessoais, criar hábitos sustentáveis e alcançar maior tranquilidade econômica.  
Com este material, você terá uma visão clara sobre como controlar gastos, planejar o futuro e evitar armadilhas financeiras comuns.

## 📖 Miniguia de Estudo

### Resumos Estruturados
- **Conceito de saúde financeira**: equilíbrio entre receitas, despesas e objetivos futuros.  
- **Controle de gastos**: registrar entradas e saídas para evitar surpresas.  
- **Planejamento de orçamento mensal**: definir limites para cada categoria de despesa.  
- **Reserva de emergência**: guardar de 3 a 6 meses de despesas essenciais.  
- **Investimentos básicos**: começar com opções seguras como Tesouro Direto ou poupança.  

### Glossário
- **Orçamento**: planejamento dos gastos e receitas mensais.  
- **Reserva de emergência**: valor guardado para imprevistos.  
- **Endividamento**: situação em que os gastos superam a capacidade de pagamento.  
- **Investimento**: aplicação de recursos com expectativa de retorno futuro.  

### Prompts Reutilizáveis
- *"Explique como montar um orçamento pessoal simples."*  
- *"Quais são os passos para criar uma reserva de emergência?"*  
- *"Liste erros comuns que prejudicam a saúde financeira."*  
- *"Sugira investimentos básicos para iniciantes."*
